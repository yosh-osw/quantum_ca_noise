# quantum_ca_noise
Data for the paper "Noise-Induced 1/f Dynamics in Cellular Automata with Quantum Probability Structures"
This file contains the results obtained from the cellular automaton simulations reported in the paper.
For each noise value, the corresponding column indicates the classification of the resulting power spectrum: when the spectrum follows a power-law distribution, the estimated exponent is reported; when it follows an exponential distribution, it is denoted as “EXP.”
Sheets 1–3 correspond to Fig. 2, Sheets 4–6 to Fig. 4, Sheets 7–10 to Fig. 6, and Sheets 11–18 to Fig. 7.
Sheet 19 reports the statistical test results described in the paper.
